# monitoring-dashboard Spec Delta

## ADDED Requirements

### Requirement: Show real MPP payment evidence

The dashboard SHALL show a real Stellar testnet payment generated by the sample PayGate MPP flow.

#### Scenario: Recipient wallet receives payment

- GIVEN the sample Express API has received a valid MPP payment
- AND the recipient wallet is entered in the PayGate dashboard
- WHEN Horizon returns the matching USDC transfer operation
- THEN the dashboard shows the payment amount
- AND increments total paid requests/payments
- AND displays the transaction hash
- AND links the hash to Stellar Expert testnet

### Requirement: Preserve SOW evidence

The project SHALL preserve enough evidence for the SOW demo and review.

#### Scenario: Evidence captured

- GIVEN an end-to-end payment succeeds
- WHEN the test is complete
- THEN the tx hash is recorded
- AND dashboard behavior is documented
- AND the demo flow can be repeated by another agent or developer
